# Руководство пользователя

## Введение

Настоящий документ содержит описание порядка взаимодействия пользователя с системой HumanGuard.

### Назначение системы
HumanGuard предназначена для защиты веб-сайтов от ботов и автоматизированных атак.

### Категории пользователей
- **Пользователь** — владелец сайта, зарегистрировавший его в системе
- **Администратор** — пользователь с правами управления всей системой

### Поддерживаемые браузеры
- Google Chrome (последние 2 версии)
- Mozilla Firefox (последние 2 версии)
- Safari (последние 2 версии)
- Edge (последние 2 версии)

## 1 Описание операций, доступных пользователю

### 1.1 Регистрация в системе

**1.1.1 Наименование операции**
Регистрация нового пользователя

**1.1.2 Условия выполнения операции**
- Отсутствуют сетевые ошибки доступа к системе
- Браузер поддерживает JavaScript и cookies

**1.1.3 Подготовительные действия**
- Установить приложение-аутентификатор на мобильное устройство (Google Authenticator, Authy)

**1.1.4 Основные действия**
1. Перейти в браузере по адресу `/auth/register`
2. Заполнить поле "Email"
3. Заполнить поле "Password" (мин. 8 символов)
4. Заполнить поле "Name"
5. Нажать кнопку "Register"
6. Отсканировать QR-код приложением-аутентификатором
7. Сохранить резервный TOTP-секрет

**1.1.5 Заключительные действия**
- Выполнено перенаправление на страницу входа
- Учётная запись создана
